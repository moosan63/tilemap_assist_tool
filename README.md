# Tilemap Assist

タイルマップ画像のタイル位置を可視化するWebツール。

itch.io等で取得したタイルマップ素材の各タイル座標を確認するために使用する。

![スクリーンショット](desc.png)

## 機能

- タイルマップ画像の読み込み（ドラッグ&ドロップ / ファイル選択）
- タイルサイズ指定によるグリッド表示（赤：縦線、青：横線）
- X/Y軸ルーラー表示
- タイルホバー時のツールチップ座標表示
- ズーム / パン操作

## 使い方

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開く。

### 操作方法

| 操作 | 説明 |
|------|------|
| ドラッグ | パン（画像の移動） |
| マウスホイール | ズームイン/アウト |
| タイルにホバー | 座標をツールチップ表示 |

## 開発

```bash
# 開発サーバー起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## 技術スタック
